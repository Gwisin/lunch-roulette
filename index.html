🔥 Firebase 연결 설정

모든 접속자가 메뉴 목록을 공유하려면
Firebase 프로젝트 설정이 필요합니다.

1단계 — console.firebase.google.com 접속
→ 프로젝트 만들기 → 이름 입력 → Google 애널리틱스 끄기 → 만들기
2단계 — 왼쪽 메뉴 빌드 → Realtime Database
→ 데이터베이스 만들기 → 지역 선택(아무거나) → 테스트 모드 선택 → 완료
3단계 — 왼쪽 메뉴 ⚙️ 프로젝트 설정
→ 아래로 스크롤 → 내 앱 → </> 웹 아이콘 클릭
→ 앱 닉네임 입력 → 앱 등록
→ firebaseConfig 코드 블록 전체 복사
※ databaseURL 이 반드시 포함되어야 합니다.
